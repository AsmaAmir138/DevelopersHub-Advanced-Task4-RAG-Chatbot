# Advanced Task 4: Context-Aware Chatbot Using Simulated RAG

This repository contains my submission for **Advanced Task 4** of the AI/ML Engineering Internship at DevelopersHub Corporation.

##  Objective
To build a conversational chatbot that utilizes Retrieval-Augmented Generation (RAG) to answer questions based on an custom vector document store, keeping track of conversation history.

##  Methodology
- **Vector Search Engine:** Implemented textual vector representations using `TfidfVectorizer` and similarity indexing through cosine similarity.
- **RAG Architecture:** Queries trigger custom search matches to pull contextual information before answering.
